# Entry-Level IT Launchpad — Start Here

A shareable job-search hub for Per Scholas graduates — and anyone who's
**CompTIA A+**-certified — breaking into tech support with limited
hands-on experience so far (for example, coming from **remote customer
service**).

## Main message

Do **not** frame this as "no tech experience." Frame it as: **CompTIA A+
certified + Per Scholas trained + remote customer service experience =
entry-level technical support candidate.** The best first roles are
people-facing tech jobs where communication matters as much as
troubleshooting.

## If you only read one thing

You are not "no experience." You are **CompTIA A+ certified + Per Scholas
trained + remote customer service**. Target **technical support, help
desk, SaaS/product support, application support, or IT asset support**.
Keep **$19/hr** as your minimum and aim for **$20-$25/hr**.

## Start here (5 steps)

- [ ] **Set your area and commute** — fill in your city/zone, ZIP,
      transportation, and commute cap.
- [ ] **Pick your target roles** — choose 2 primary + 1 backup from the
      Best-Fit Roles & Pay page.
- [ ] **Set up your search** — copy the search strings and job-alert
      links from the Search Toolkit for your zone.
- [ ] **Update your resume and LinkedIn** — use the Resume, LinkedIn &
      Cover Letter page.
- [ ] **Apply and track** — use the 2-week plan and weekly tracker in
      the Application Plan page.

## Location note

This guide uses Houston as a worked example metro. Anywhere you see
`Houston`, swap in your own city. Houston is huge and car-dependent, so
the Houston zones page breaks it into areas and covers transportation.
For fully remote roles, use the remote searches instead — remote removes
the commute entirely.

## Bottom line

The strongest first targets are **Remote Technical Support Specialist**,
**Help Desk Analyst I**, **SaaS/Product Support Specialist**,
**Application Support Analyst**, and **IT Asset Coordinator**. Keep
$19/hr as the minimum, aim for $20-$25/hr for hourly roles, and use Per
Scholas referrals whenever possible.
